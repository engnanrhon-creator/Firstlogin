# AQUASAVE (Integration)

Firmware and hardware integration source code for the AQUASAVE IoT-based aquaponics monitoring system. This repository contains ESP32 controller logic, sensor integration, real-time data acquisition, automation features, and communication with Firebase for remote monitoring.

Developed firmware for ESP32-based main controller. 
 - Integrated multiple sensors: pH Sensor Turbidity Sensor Water Level Sensor Temperature Sensor Humidity Sensor
 - Implemented real-time sensor data collection and processing.
 - Connected ESP32 to Firebase for cloud data synchronization.
 - Developed automated monitoring and alert functionalities.
 - Supported fish feeding automation and grow light control.
 - Implemented hardware-software communication and device management.
 - Performed sensor calibration, testing, and troubleshooting.
